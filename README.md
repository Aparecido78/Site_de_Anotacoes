# 📝 Site de Anotações

Aplicação full stack para gerenciamento de anotações pessoais.

O usuário pode criar conta, fazer login e gerenciar suas anotações de forma segura.

---

# 🚀 Tecnologias

## Back-end
- Node.js
- Express
- Prisma ORM
- MySQL
- JWT (Autenticação)

## Front-end
- React.js
- TypeScript
- CSS

## Bibliotecas
- bcrypt
- jsonwebtoken
- multer
- jwt-decode
- axios
- react-toastify

---

# ⚙️ Funcionalidades

✔ Cadastro de usuários  
✔ Login com autenticação JWT  
✔ Criar anotações  
✔ Editar anotações  
✔ Excluir anotações  
✔ Listar anotações do usuário  
✔ Edição de senha  
✔ Edição de dados pessoais  
✔ Edição de contatos  

---

# 🔐 Segurança

- Senhas criptografadas com **bcrypt**
- Autenticação via **JWT**
- Rotas protegidas no backend
- Validação de usuário nas anotações
